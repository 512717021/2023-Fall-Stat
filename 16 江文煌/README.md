# 研究方法
## 量比價先行
### 以ㄧ檔0050成份股為研究標的，取一整個年度的交易資料。分析是否當成交量有相當變化幅度後，在某個時間點，會有相當幅度的價格變化幅度，而且此現象會存在非常高的出現機率。                    

# 研究變數定義與目標
## 研究標的: 2330.tw 台積電
資料區間(一年)：2022.01.03~2022.12.30(交易日)
成交量變化幅度: 50% up
價格變化幅度: 2% up
出現機率(重現率): 90% up
